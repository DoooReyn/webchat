# mingchat
nodeJs 在线聊天室.    
   
[在线地址](http://mingchat.herokuapp.com/)
  
![demo](description.png)  
    
功能说明：   
1. 切换文字颜色;    
2. 发送表情;   
3. 切换背景图片;  
4. 所在城市显示;    
5. 等级积分制;   
6. 昵称或匿名;
